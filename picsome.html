Pic-Some

github | live link

Built with: JSX, CSS, REACT.JS

Pick your favourite poster and get it delivered directly to your home. You can add a little heart icon in the upper left corner and add it to cart in the upper right corner. To actually finalise your order go to the cart page, check if everything is ok, and then place your order.

Playing with API's is always interesting. Getting the data and combining it with the Context API, React Router, creating the icon functionality was a blast. And Hooks are a very solid addition to the React library as well. Once you get used to them there is no going back (except for some old projects :-P ).
